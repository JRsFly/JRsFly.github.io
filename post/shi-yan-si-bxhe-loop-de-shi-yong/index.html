实验四[bx]和loop的使用

1）编程，向内存0:20000:23F依次传送数据063(3FH)

2）编程，向内存0:20000:23F依次传送数据063(3FH)，但是只可以用9条指令，包括“mov ax,4c00h”和"int 21h"

我把第一题和第二题一起做了，所以就只写了一个源程序

分析：使用ds段地址变量来实现传送数据，同时bx即作为偏移地址也作为数据来使用

assume cs:code
code segment
	mov ax,20H
	mov ds,ax		;ds=20H
	
	mov bx,0		
	mov cx,64		;循环执行64次

s:	mov [bx],bx		;向ds:bx中存入bx的值
	inc bx
	loop s
	
	mov ax,4c00h
	int 21h
code ends

end

用debug进行调试

然后再用t命令一步步执行语句或者直接用-g命令

这是我用-t命令一条条调试后的结果

如上图可以把0:2000:23F存入数据063(3FH)，由于单步执行次数过多，剩下的就用-g命令了

3）下面的程序功能是将"mov ax,4c00h"之前的指令复制到内存0:200处，补全程序。上机调试，跟踪运行结果

assume cs:code
code segment
	mov ax,____
	mov ds,ax
	mov ax,0020h
	mov es,ax
	mov bx,0
	mov cx,____
s:	mov al,[bx]
	mov es:[bx],al
	inc bx
	loop s
	mov ax,4c00h
	int 21h
code ends
end

分析：显然可以看出将指令复制到0：200，通过的是把指令段地址cs通过ax传递给ds，所以第一个空填入cx，第二个空通过上机debug调试即可得到循环进行次数

所以要把"mov ax,4c00h"之前的指令复制到内存0：200处，cx就等于0017
